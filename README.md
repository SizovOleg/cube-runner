# 🟩 Cube Runner: Battle Dash 🚀

**Cube Runner: Battle Dash** — это динамичный 2D-платформер и автораннер, написанный на **React + TypeScript + Canvas API**. Игра развивает концепцию классического *Geometry Dash*, добавляя элементы шутера, прокачку, бои с боссами и разнообразные игровые механики.

## ✨ Особенности и Механики

Игра полностью переработана визуально и геймплейно, предлагая **5 уникальных уровней** (от "Training Grounds" до "Digital Core"), каждый из которых наполнен своими механиками и эстетикой:

*   **⚡ Скорость и Ритм (Speed Portals)**: Динамическое изменение скорости игры с помощью порталов (Slow, Normal, Fast, Superfast), меняющих темп прямо на лету (вдохновлено *Geometry Dash*).
*   **🌌 Искажение Гравитации (Gravity Portals)**: Порталы бросают вызов гравитации, заставляя кубик бежать по потолку, полностью меняя физику и визуализацию прыжков и полётов.
*   **💨 Воздушная Акробатика (Jump Rings / Orbs)**: Светящиеся сферы в воздухе (Жёлтая — обычный прыжок, Розовая — малый прыжок). Требуют точного клика/нажатия для совершения прыжка прямо в воздухе.
*   **🚀 Батуты (Jump Pads)**: Автоматически подбрасывают игрока на разную высоту при наступании (Красные — супер-высоко, Жёлтые — стандартно, Розовые — низко).
*   **💥 Разрушаемые блоки и Лазеры**: Ловушки, требующие идеального тайминга. Платформы могут рассыпаться под кубиком, а прерывающиеся лазерные лучи моментально сжигают.
*   **👹 Эпичные Боссы**: В конце каждого уровня вас ждёт босс (Mech, Serpent, Sentinel, Inferno, Core) с уникальными фазами атаки.
*   **🛒 Магазин и Прокачка**: Собирайте монеты на уровнях, чтобы разблокировать **6 уникальных скинов** (каждый оставляет свой неоновый след) и покупать **улучшения/Powerups** (Магнит, Замедление времени, Двойной прыжок, Ракеты, Щиты).

## 🛠 Технологический стек

*   **Frontend**: React 18, TypeScript
*   **Rendering**: HTML5 Canvas API (кастомный 2D движок)
*   **Build Tool**: Vite
*   **Стилизация**: CSS/Inline Styles + Glassmorphism эффекты, неоновая пульсация.
*   **Архитектура движка**:
    *   `src/engine/` — Ядро: рендер (`Renderer.ts`, `BackgroundRenderer.ts`), Game Loop, Физика (`Physics.ts`), Ввод (`Input.ts`), Частицы (`ParticleSystem.ts`).
    *   `src/entities/` — Классы сущностей (`Player.ts`, `Enemy.ts`, `Boss.ts`, `Powerup.ts`, `Bullet.ts`).
    *   `src/levels/` — Данные всех 5 уровней, парсер препятствий и управление прогрессом.
    *   `src/ui/` — React-компоненты для всех меню (Выбор уровня, Магазин, Экраны смерти/победы).

## 🚀 Как запустить локально

Вам потребуется установленный [Node.js](https://nodejs.org/).

```bash
# Клонировать репозиторий
git clone https://github.com/SizovOleg/cube-runner.git

# Перейти в папку
cd cube-runner

# Установить зависимости
npm install

# Запустить dev-сервер
npm run dev
```

Откройте `http://localhost:5173/` (или другой выданный Vite порт) в вашем браузере.

## 🎮 Управление

| Действие | Клавиатура | Сенсорный экран / Мышь |
| :--- | :--- | :--- |
| **Прыжок / Полёт (Коридор) / Клик по Сфере** | `Space`, `W`, `↑` | Клик левой кнопкой мыши / Тап |
| **Стрельба** | `X`, `Z`, `Shift` | Кнопка «GUN» на экране |
| **Суперспособности (Powerups)** | `1`, `2`, `3` | Кнопки слотов 1-3 |
| **Пауза** | `Esc`, `P` | — |
